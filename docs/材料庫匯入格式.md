# 材料庫匯入格式

Phase 1「匯入材料庫」接受 JSON 檔，供大量載入公司內部整理的材料係數表。
**資料檔本身保存於內部儲存空間，不隨工具程式碼發佈**；匯入後存於瀏覽器
localStorage（隨專案檔一併匯出/匯入），與內建參考庫合併顯示於材料下拉選單。

## 檔案格式

```json
{
  "formatVersion": 1,
  "materials": [
    {
      "name": "低合金鋼 調質 HB300",
      "kc1": 1725,
      "mc": 0.24,
      "isoGroup": "P",
      "gammaRef": 0,
      "ffRatio": 0.4,
      "fpRatio": 0.3,
      "source": "（來源說明）",
      "verified": false
    }
  ]
}
```

亦接受純材料陣列（不含外層物件）。

## 欄位說明

| 欄位 | 必填 | 型別 | 說明 |
|:--|:--:|:--|:--|
| `name` | ✔ | string | 材料名稱（匯入時同名更新、新名附加） |
| `kc1` | ✔ | number > 0 | 比切削力參考值 [N/mm²]（h = 1 mm 條件） |
| `mc` | ✔ | 0 < number < 1 | 比切削力指數 |
| `isoGroup` | | P/M/K/N/S/H | ISO 切削材料群，缺省 P |
| `gammaRef` | | number | kc1 測定基準前角 [°]；缺省 6（Sandvik 體系）；Kienzle kc1.1 體系填 0 |
| `ffRatio` | | number > 0 | 進給分力比 Ff/Fc，缺省依引擎預設 0.40 |
| `fpRatio` | | number > 0 | 背分力比 Fp/Fc，缺省依引擎預設 0.30 |
| `source` | | string | 資料來源說明，缺省空字串 |
| `verified` | | boolean | 是否已核對現行廠商手冊，缺省 false（UI 顯示「須核對」） |
| `id` | | string | 缺省自動產生；同名更新時保留既有 id |

驗證失敗的資料列會被略過並於 UI 列出原因（含列號），不影響其他列匯入。

## 匯出

Phase 1「匯出材料」按鈕可將目前的自訂材料輸出為同格式 JSON，便於在內部分享或備份。
