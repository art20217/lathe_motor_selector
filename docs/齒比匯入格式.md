# 齒比匯入格式

Phase 3「匯入齒比」接受 JSON 檔，一次載入四檔齒比與傳動效率
（例如齒輪箱機構設計完成後，將實際齒數比回填驗證，即 SOP Step 3.5）。

## 檔案格式

```json
{
  "formatVersion": 1,
  "gears": [
    { "ratio": 0.0025, "efficiency": 0.85 },
    { "ratio": 0.0089, "efficiency": 0.88 },
    { "ratio": 0.0238, "efficiency": 0.91 },
    { "ratio": 0.0846, "efficiency": 0.94 }
  ]
}
```

亦接受純陣列（不含外層物件）。

## 規則

| 項目 | 說明 |
|:--|:--|
| 筆數 | 恰為 4 筆（四檔變速箱固定），依序對應第 1–4 檔 |
| `ratio` | 齒比 i = n_sp / n_motor，須為正數（減速傳動 < 1） |
| `efficiency` | 該檔傳動效率，0 < η ≤ 1 |
| 驗證 | 任一檔驗證失敗即整檔拒收（避免部分套用造成檔位錯位），原因於 UI 列出 |

## 匯出

Phase 3「匯出齒比」可將目前四檔設定輸出為同格式 JSON，便於在專案間複用或內部分享。
